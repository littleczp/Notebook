> 国家的兴衰
>
> * 社会较稳定时（规模和边界不变），随着时间的推移，将出现愈来愈多的集团和组织
> * 小团体组织集体行动的能力强，但这种优势**随着社会稳定时间的延长而递减**
> * 特殊利益集团会降低社会效率和总收益，并使政治的分歧加剧
> * 代表的利益愈广泛，该组织或集团会更倾向于增加社会总收益
