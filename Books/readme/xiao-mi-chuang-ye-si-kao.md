软件是一个工具，互联网提供的是服务。（那AI带来的是什么？是能力，给出方案并完成）



有所为，有所不为：决定不做什么和决定做什么一样重要

> "用户第一"、“以用户为中心”不是使命，也不是愿景

使命：企业存在的目的和意义

愿景：基于使命，企业最终希望成为的样子



极致不是自嗨和自我感动，是接受更长远的"延迟满足感"

1. 追求的极致是不是用户真正需要的？
2. 追求的极致能不能成为核心竞争力？
3. 追求的极致能不能形成长期可持续的竞争壁垒？



好产品不一定带来口碑；便宜的产品不一定带来口碑；又好又便宜的产品也不一定带来口碑。

{% hint style="warning" %}
只有超过预期的产品才能带来口碑
