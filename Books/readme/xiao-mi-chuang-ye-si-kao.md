## 专注

有所为，有所不为：决定不做什么和决定做什么一样重要

> "用户第一"、“以用户为中心”不是使命，也不是愿景

使命：企业存在的目的和意义

愿景：基于使命，企业最终希望成为的样子



## 极致

极致不是自嗨和自我感动，是接受更长远的"延迟满足感"

1. 追求的极致是不是用户真正需要的？
2. 追求的极致能不能成为核心竞争力？
3. 追求的极致能不能形成长期可持续的竞争壁垒？



## 口碑

好产品不一定带来口碑；便宜的产品不一定带来口碑；又好又便宜的产品也不一定带来口碑。

{% hint style="warning" %}
只有超过预期的产品才能带来口碑

预期是一个相对值，每一个消费者在购买时都会有一个价值平家表现的心理预估。
{% endhint %}



## 快

一家更快的公司，有更高的效率、更高的洞察力、更好的行业适应性、更坚韧的生命力和更持续的创新能力

{% hint style="info" %}
洞察快、响应快、决策快、改善快
{% endhint %}
