# 风口

未来5到10年，最大的风口不是某个具体的行业，而是社会分工逻辑的彻底重构

> 过去三十年，我们的财富积累逻辑是增量逻辑。只要你敢冲，快慢不重要，反正都能到终点。
>
> 但现在，增量时代彻底结束了，存量博+不是简单的内卷，而是残酷的优胜劣汰。

{% stepper %}
{% step %}
### 白领阶层的消亡与超级个体的崛起

{% hint style="warning" %}
过去我们认为的高级脑力劳动，在算力成本无限趋近于零的今天，变得一文不值。

以前一个团队十个人干的活，现在一个人加三个Agent就能干完。这导致中间管理层会发生断崖式的坍塌。

\
**未来的风口在哪里?在于你能否成为那个驾驭Agent的人。**&#x5728;于你能不能把你现在的行业经验，转化为一套可以被AI执行的标准化SOP，然后自己出来单干。
{% endhint %}
{% endstep %}

{% step %}
### 无法被数字化的，将变得极度昂贵。

{% hint style="info" %}
线上内容越来越泛滥，越来越廉价。短视频、网文、合成的音乐等娱乐内容边际成本已经是零。人类在数字世界的快感阈值被拉得无限高，但内心却越来越空虚。

\
于是，一个巨大的反向风口出现了：此时此刻，此情此景，最真实的体验。\
越是AI发达，人越是渴望真实的触碰。所以未来十年，顶级的服务业将迎来爆发。
{% endhint %}
{% endstep %}
{% endstepper %}
