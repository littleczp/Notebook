# 风口

未来5到10年，最大的风口不是某个具体的行业，而是社会分工逻辑的彻底重构

> 过去三十年，我们的财富积累逻辑是增量逻辑。只要你敢冲，快慢不重要，反正都能到终点。
>
> 但现在，增量时代彻底结束了，存量博弈不是简单的内卷，而是残酷的优胜劣汰。

{% stepper %}
{% step %}
### 白领阶层的消亡与超级个体的崛起

{% hint style="warning" %}
过去我们认为的高级脑力劳动，在算力成本无限趋近于零的今天，变得一文不值。

以前一个团队十个人干的活，现在一个人加三个Agent就能干完。这导致中间管理层会发生断崖式的坍塌。

\
**未来的风口在哪里?在于你能否成为那个驾驭Agent的人。**&#x5728;于你能不能把你现在的行业经验，转化为一套可以被AI执行的标准化SOP，然后自己出来单干。
{% endhint %}
{% endstep %}

{% step %}
### 无法被数字化的，将变得极度昂贵

{% hint style="info" %}
线上内容越来越泛滥，越来越廉价。短视频、网文、合成的音乐等娱乐内容边际成本已经是零。人类在数字世界的快感阈值被拉得无限高，但内心却越来越空虚。

\
于是，一个巨大的反向风口出现了：此时此刻，此情此景，最真实的体验。\
越是AI发达，人越是渴望真实的触碰。所以未来十年，顶级的服务业将迎来爆发。
{% endhint %}
{% endstep %}

{% step %}
### 情绪价值与精神出口

{% hint style="danger" %}
现在社会的抑郁症、焦虑症发病率是多少?那是天文数字。物质过剩的年代，精神危机是必然产物。



未来5到10年，心理咨询、身心灵疗愈、陪伴经济、甚至是新兴的宗教形态，都将迎来井喷。\
任何行业，只要能给用户提供情绪价值，都能重做一遍。
{% endhint %}
{% endstep %}

{% step %}
### 硬科技与出海

> 未来十年，中国最重要的任务只有两个：一个是安全，一个是走出去。
>
>
>
> 至于走出去，不是简单的卖货。过去的跨境电商是把中国便宜的小商品卖给全世界，赚个差价。未来的出海，是供应链的出海，是品牌的出海，甚至是文化模式的出海。

真正的风口，永远在阻力最小的方向，而不在竞争最激烈的地方。
{% endstep %}

{% step %}
### 阶层的K型分化与生存策略

{% hint style="success" %}
来5到10年，中产阶级会面临前所未有的清洗。上面的人通过掌握核心资产和核心算法，财富呈指数级增长；下面的人通过接受廉价的娱乐和基础保障，维持生命。最难受的是中间这拨人。



在K型分化的社会里，只有两个方向能赚钱。\
要么，你做极其高端的生意，服务那1%的富人。他们对价格不敏感，对品质和体验极度挑剔。这需要你有极高的审美和圈层资源。\
要么，你做极致性价比的生意，服务那99%的大众。像拼多多一样，把价格打到地板，靠规模效应和极致的供应链管理赚钱。
{% endhint %}
{% endstep %}
